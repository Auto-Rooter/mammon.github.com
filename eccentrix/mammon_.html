Projects
The Bastard
A UNIX disassembler. Currently handles ELF and a.out files on Linux. Under perpetual development ... expect a 1.0 release from my retirement villa.

Libdisasm
A library incorporating the x86 disassembler from The Bastard. Written in straight C to compile on Linux, UNIX, and Win32 platforms. Freedom from the tyranny of libopcodes at last!

The Dude
A Linux process debugger that doesn't use ptrace(1). That's right, does not. The ptrace(1) interface is unreliable, easy to fool and insecure to boot. Friends don't let friends use ptrace(1). Of course, given the state of the project, friends don't let friends use the-dude either...

UNIX on the X86-64 Laptop
What should have been a simple install has turned into a huge struggle to configure and port various OS applications and drivers. Consider this an x86-64 laptop review and a how-to guide.


Tales
These are pretty old and are not maintained, but for some reason people still like to read them. The assembly language one in particular seems to be in vogue at universities.

ASM Intro
Titled "Registers, Memory, and how Assembly Language came to Mankind". I guess this one won't be out of date until IA-64 takes over.

RE Intro
Titled "Of The Engine Reversi", this pretends to provide some basic introduction to reverse engineering. Most of it is a wash, but the crackme for some group somewhere had some good SMC in it, and it is taken apart here.

SoftICE Intro
Titled "Mankind comes into the Ice Age", this is from the 2.6-3.0 days. Use at own risk.

Disassembler Intro
Titled "Breathing Life into Dead Listings", this is essentially a survey/tutorial of the use of disassemblers available in 1997 or 1998.

IDA Intro
Variously titled "Using IDA", "The Great IDA Primer", etc; this grew out of a long post on Fravia's old --and now long-dead-- MB. There may be a better-looking version with Fravia's notorious '+' markups at searchlores.org.

Long-dead Projects
Pointless projects from the mid-90's. People seem to like the screensaver and notepad.exe ones.

PE File Format
Everything you never wanted to know about the Win32 PE file format, circa Win 95 and NT4.

Linux Tale 1: Beginning
Titled "Scalable Demand-Paged World Domination", this contains what every newbie should do to their system after install. Unfortunately this dates to the kernel 2.2 days.

Linux Tale 2: Advanced
Titled "32-bit Task-Switching Synchronized Insurrection", this covers more advanced topics, including network stuff, useful tools, and how to write programs for ptrace(1) and libopcodes.

Linux Tale 3: Shell Scripting
Titled "Straight to Shell", this is an unfinished tutorial on shell scripting. As with the rest of the tales, it is long and quite boring.

Page Faults

Don't take this article too seriously, it is part of the 'reality cracking' movement which, if it is not already a joke, should be.

Decompiling a string manipulation routine
A case study in decompiling assembly language to C code. Actually, someone sent me this code with a question mark at the end, and I wrote this up as a response, then incorporated it into a reverse engineering site I was working on at the time.

Reading Files into Linux Kernel Memory
A very crude, very hackish way of reading files while in kernel mode. Typical reactions were stuff like "them: 'why don't you just use the get_user() method everybody else has been using?' me: 'cause that's the wanker way.'"

Hooking Interrupt and Exception Handlers in Linux

The Dude Paper
A write-up on the ideas behind the-dude debugger and -lduderino. Not intended for casual reading ... in fact, only of interest to people with the sick ambition of working on or with the-dude.

"Mammon_'s Guide to Writing Disassemblers in Perl, You Morons!"
