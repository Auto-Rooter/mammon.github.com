-{_____ The Dude_____________________________________________________________}-

The Dude Project

The Dude Page

The Dude Forum

The Dude Paper

The DUDE is a replacement debugging interface for UNIX, intended to overcome the failings of ptrace(2) when debugging third-party or hostile code. The package consists of

  • the_dude.o: a kernel module providing the core debugging support
  • libduderino.so: a user-space library wrapping the debugging facility in an API
  • ddbg: an ncurses debugger demonstrating the use of the library

    Currently only a Linux implementation is provided. Project is in early Alpha stages. Note: Requires libdisasm from the bastard project.



    -{_____ SecPtrace_____________________________________________________________}-

    Readme

    Source Code

    A quick hack to secure ptrace() so that only authorized processes can call it. All calls to ptrace() still return success, so this is useful for foiling the old ptrace()-based anti-debugging trick.
