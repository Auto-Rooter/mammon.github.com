Win32 Assembly Programming Toolkit


What follows is a collection of free development tools for win32 assembly language programming. Individual licenses for each of the packages may or may not allow commercial development; however academic, personal, and non-profit development are permitted. Each of the packages below has been evaluated and found to be suitable for recommendation to both the beginner and the accomplished assembly language programmer.

ALIB: LIB file generator for ALINK [Homepage]

ALINK: 32-bit linker [Homepage]

BIEW: Binary File Editor [Homepage]

Borg: Win32 Disassembler [Source]

GoRC: Win32 Resource Compiler [Homepage]

GoVEST: Win32 GUI-mode Debugger

NASM: 32-bit Assembler

QEditor: Programmer's editor [Homepage]

Win32.lib: Win32 API lib files for ALINK [Homepage]
