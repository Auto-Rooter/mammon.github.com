GoVest! Release Notes
=====================

GoVest! is a Win32 debugger and disassembler. It is capabable of using
source code information in COFF, PDB and VOM format.

DISCLAIMER: GoVest! is provided as is. Use at your own risk. 
GoVest! is freeware, you may distribute it as long as all the files in the zip 
are in their original form.

The current GoVest! version is 0.9beta. Sorry, there is currently no online help
file. Check the GoVest! homepage at

http://ourworld.compuserve.com/homepages/atrimborn

for more details, newer versions and the GoVest! FAQ.

July 1999, Ansgar Trimborn
